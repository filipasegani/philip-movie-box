# Philip Movie Box

A premium movie streaming application inspired by Netflix.

## Features
- Modern, dark-themed UI
- Responsive design
- Movie categorization (Trending, Top Rated, Action, etc.)
- Hero section with video playback simulation

## Tech Stack
- React
- Vite
- Vanilla CSS
- Lucide React (Icons)

## Setup Instructions

1.  **Install Node.js**: Download and install from [nodejs.org](https://nodejs.org/).
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
4.  **Build for Production**:
    ```bash
    npm run build
    ```
